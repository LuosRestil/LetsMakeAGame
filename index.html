Let's Make A Game

Welcome

Hello, and welcome to Let's Make A Game, where we'll make "Rudolph vs The Blobs", a totally original game that definitely no one has ever thought of before, from the ground up, using HTML Canvas and TypeScript. This game was created for a winter holiday game jam, and so is winter holiday themed, and is scoped to be completed in a short time period with minimal prior experience. This project is intended for someone with some web programming experience who just hasn't made a game before. We'll be writing every line of code together, but we won't be going over the basics of programming in general or HTML/CSS/TypeScript in particular. We will be going into detail about game-specific programming concepts as they arise in our project, like animation/physics, vectors, particle systems, et cetera. The completed code for each lesson can be found in its own branch on GitHub. See you in the first lesson!
